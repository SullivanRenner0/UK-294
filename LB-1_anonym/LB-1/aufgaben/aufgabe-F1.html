To Do:
Beantworten Sie folgende Fragen zum Thema Sicherheit
  1. Nennen Sie mindestens 5 Möglichkeiten, um einen Zugriff zu schützen
  2. Beschreiben Sie diese 5 Möglichkeiten auf der rechten Seite jeweils kurz und einfach - was verstehen Sie unter den jeweiligen Begriffen (verwenden Sie Ihre eigenen Worte und beschreiben Sie es so, als würden Sie es einer Person ohne grosse Kenntnisse erklären)?
Kompetenz:
  • Zugriff schützen
Ihre Lösung:
  1. HTTPS : Verschlüsselt den Datenaustausch zwischen dem Client und dem Server
  2. JWT : Wenn man sich einloggt kriegt man ein Token und kann nur mit diesem Token CRUD-Funktionen ausführen
  3. 2FA : 2-Faktor-Authentifizierung, neben dem Password muss man, z.B., auch ein Code von einer SMS angeben
  4. Server Sessions : Der Server speichert Sessions und eine Benutzername + Passwort Kombi kann nur einmal gleichzeitig verwendet werden.
  5. SOP : Same-Origin-Policy, das HTML für eine Webseite + die Daten können nur vom gleichen Ort geladen werden.

Aufgabe F1: Sicherung des Zugriffs 5 Pkt.

To Do:
Beantworten Sie folgende Fragen zum Thema Sicherheit
  1. Nennen Sie mindestens 5 Möglichkeiten, um einen Zugriff zu schützen
  2. Beschreiben Sie diese 5 Möglichkeiten auf der rechten Seite jeweils kurz und einfach - was verstehen Sie unter den jeweiligen Begriffen (verwenden Sie Ihre eigenen Worte und beschreiben Sie es so, als würden Sie es einer Person ohne grosse Kenntnisse erklären)?
Kompetenz:
  • Zugriff schützen
Ihre Lösung:
  • Session IDs
  • HTTPS
  • 2 Faktor Authentifizierung
  • Passwort Mindestanforderungen
  • SOP

Session IDs: Der Benutzer bekommt eine eigene Session, mit der alle Anfragen auf den Server gekennzeichnet werden. So bekommt nur er die Daten, die er abfragt.


HTTPS: Ist die sichere Version von HTTP. Sie verschlüsselt die Serveranfragen.


2FA: Beispielsweise ein Code auf dem Handy per SMS, wenn man sich einloggen will.


Passwort Mindestanforderungen: Beispielsweise mindestens 12 Zeichen, keine Wörter etc


SOP: Bedeutet Same-Origin policy. Es schaut, mit welcher IP sich der User anmeldet, und sendet dann nur Daten dort hin.
